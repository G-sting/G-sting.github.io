python判断语句和循环控制

什么是流程

  • 计算机执行代码的顺序

什么是流程控制

-对计算机代码执行的顺序进行有效的流程管理，只有流程控制才实现在开发过程中的业务逻辑

流程控制的分类：

  • 顺序流程：代码一种自上而下的执行结构，也是python的默认流程

  • 选择流程/分支流程：根据在某一步的判断有选择的去执行相应的逻辑的一种结构

    • 单分支
      if 条件表达式：比较运算符/逻辑运算符/符合表达式
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......
    • 双分支
      if 条件表达式：比较运算符/逻辑运算符/符合表达式
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......
      else :
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......
    • 多分支
      if 条件表达式：
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......
      elif 条件表达式:
      一条条的python代码
      一条条的python代码
      一条条的python代码
      elif 条件表达式:
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......
      else :
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......
      特征：只要满足其中一个分支，就会退出本层if结构【必定会执行其中一个分支】

  • 至少有两种情况可以选择

  • elif 后面必须跟上条件和语句
    -else是选配，根据实际情况来填写
    条件表达式：比较运算符/逻辑运算符/复合的运算式
    if-else嵌套结构

  • 一个场景需要分阶段或者层次，做出不同的处理

  • 需要执行内部的if语句，首先要满足外部的if语句才可以

  • 循环流程：在满足一定的条件下一直重复的去执行某段代码的逻辑【事情】

    • while 条件表达式：
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......

    • 循环必须有一个初始值

    • 条件表达式

    • 变量【循环体内计数变量】的自增或者自减，否则会造成死循环

    • 使用条件：循环次数不确定，是依靠循环条件来结束

    • 目的:为了将相似或者相同代码操作变得更加简洁，使得代码可以重复利用

    • for ... in 可迭代集合对象：
      一条条的python代码
      一条条的python代码
      一条条的python代码
      ......
